Es gibt ein Problem
Irgendwas ist schief gelaufen. Wahrscheinlich versuchen Sie Koto-Player auf einem Gerät auszuführen, dass nicht webOS 2.0 installiert hat. Wenn Sie für Koto-Player bezahlt haben, tut es mir leid. Allerdings wird in der Beschreibung der App ausdrücklich darauf hingewiesen, dass sie nur mit webOS 2.0 funktioniert. Weder HP noch ich werden das Geld zurück erstatten.
So wie es aussieht sind Sie vom Palm-media-indexer-Fehler betroffen. Wenn Sie sich mit webOS Quick Install oder Preware nicht auskennen, laden Sie webOS Quick Install herunter und folgen Sie dabei diesen Anweisungen: URL.

Sobald Sie die Möglichkeit haben .ipks zu installieren, laden Sie das folgende .ipk herunter und installieren Sie es. Dadurch sollte der Fehler behoben werden: IPK herunterladen.

Bei Problemen, können Sie uns gerne eine E-Mail schicken: kotoplayer@gmail.com.
